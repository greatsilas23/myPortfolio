##ComplimentaryEquation
A web component that allows users to input roots of auxiliary equation from a differential equation and get a complimentary equation based on whether the roots of the auxiliary equation are
<ol>
    <li>real and distinct roots</li>
    <li>complex conjugate roots</li>
    <li>real and equal roots</li>
</ol>
